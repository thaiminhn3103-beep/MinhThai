# Đồ án Quản lý Sinh viên (VB + SQL Server)

## Chức năng
- Thêm, sửa, xóa sinh viên
- Quản lý lớp học
- Kết nối SQL Server

## Cách chạy
1. Import file `qlsv.sql` trong SQL Server để tạo database.
2. Mở `QLSV_vb.sln` bằng Visual Studio.
3. Chỉnh sửa chuỗi kết nối SQL trong code nếu cần.
4. Run project.
